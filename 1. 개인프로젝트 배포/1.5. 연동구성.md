### (백엔드) CORS 설정을 추가함

```java
        registry
                .addMapping("/**") // 모든 경로에 대해
                .allowedOrigins(
                        "http://localhost:5500",
                        "http://127.0.0.1:5500",
                        "http://ec2-3-34-159-235.ap-northeast-2.compute.amazonaws.com")
                .allowedMethods("*"); // 모든메서드
```

### (백엔드) 프론트엔드와 연동되는 URL, 도메인등을 변경

```
# application.properties
app.cookie.domain=localhost
app.login.url=http://localhost:5500/login.html
app.home.url=http://localhost:5500

```

```
# application-dev.properties
app.cookie.domain=ec2-3-34-159-235.ap-northeast-2.compute.amazonaws.com
app.login.url=http://ec2-3-34-159-235.ap-northeast-2.compute.amazonaws.com/login.html
app.home.url=http://ec2-3-34-159-235.ap-northeast-2.compute.amazonaws.com

```

```java
@Value("${app.cookie.domain}")
private String cookieDomain;

@Value("${app.login.url}")
```
