## Build Steps

- Execute Windows batch command

```shell
#jar 빌드
echo "-------BUILD JAR ----------------"
./gradlew bootJar
```

```shell
#이미지 빌드 및 푸시
echo "-------BUILD IMAGE AND PUSH ----------------"
docker build -t 도커허브계정명/이미지명 .
docker login -u 도커허브계정명 -p 도커허브패스워드
docker push 도커허브계정명/이미지명
```
